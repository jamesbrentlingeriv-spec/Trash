# Stop the Rumpke Expansion - Montgomery County

A campaign website to raise awareness about the Rumpke Landfill expansion
proposal.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Run the app: `npm run dev`
